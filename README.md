<h1 align="center">🧮 Math Quiz Game (C++)</h1>

<p align="center">
  A simple and interactive math quiz game built using C++.<br>
  Practice arithmetic operations in a fun way!
</p>

<hr>

<h2>📌 Features</h2>

<ul>
  <li>🎯 Multiple difficulty levels (Easy, Medium, Hard, Mixed)</li>
  <li>➕➖✖️➗ Supports all basic math operations</li>
  <li>🔀 Random question generation</li>
  <li>📊 Tracks correct and wrong answers</li>
  <li>✅ Displays correct answer when wrong</li>
  <li>🏁 Final result summary (Pass / Fail)</li>
</ul>

<hr>

<h2>🎮 How It Works</h2>

<ol>
  <li>Choose number of questions</li>
  <li>Select difficulty level</li>
  <li>Select operation type</li>
  <li>Answer the questions</li>
  <li>See your final score</li>
</ol>

<hr>
